# Banking System

A C++ program that simulates a basic banking system with account management.

## Features
- Deposit money into account
- Withdraw money from account
- Transfer funds between accounts
- View account balance
- Transaction history tracking
- Display account details to customers

## Classes Used
- Account class with owner, balance, transactions
- Transaction struct for history

## How to Run
1. Clone this repository
2. Open terminal and navigate to folder
3. Compile: g++ banking.cpp -o banking
4. Run: ./banking

## Technologies Used
- C++
- Object Oriented Programming (OOP)
- Standard Template Library (STL)
